## üìñ Quick Start Guide

2. Enter a vault name (e.g., "Personal Passwords")

### Adding Password Entries

1. Open your vault by clicking on it
2. Click **"Add Entry"** button
3. Fill in the details:
   - Title (e.g., "Gmail Account")
   - Username/Email
   - Password (or generate one with the password generator)
   - URL (optional)
   - Notes (optional)
4. Click **"Save"** - entry is encrypted and stored securely

### Generating Secure Passwords

1. Click the **"Password Generator"** button when adding/editing entries
2. Customize settings:
   - Length: 8-128 characters
   - Include uppercase, lowercase, numbers, symbols
   - Enable memorable mode for easier recall
3. Click **"Generate"** - copy and use your new secure password

### Creating Backups

1. Navigate to **"Backup"** page from the sidebar
2. Click **"Create Backup"** button
3. Choose backup location (local folder, USB drive, network share)
4. Backup is encrypted and verified automatically
5. Schedule automatic backups for hands-free protection

### Enabling Stealth Protection

1. Go to **"Stealth Manager"** from the security menu
2. Click **"Protect Files"** button
3. Select files or folders to hide
4. Choose protection level (Hide, Stealth, or Steganography)
5. Files become invisible to standard file explorers
6. Use PowerVault to reveal or permanently restore files

## üîê Security Architecture

### Encryption Standards

**Data at Rest:**
- **AES-256-GCM**: Authenticated encryption with Galois/Counter Mode
- **PBKDF2**: 500,000 iterations for key derivation
- **Random Salt**: Unique per-vault cryptographic salt
- **IV (Initialization Vector)**: Random 12-byte nonce per encryption operation

**Data in Transit (Backups):**
- Same AES-256-GCM encryption applied to all backup files
- SHA-256 hashing for integrity verification
- Tamper detection with authentication tags

**Memory Security:**
- Automatic secure memory clearing after use
- Protected memory regions for sensitive operations
- No plaintext passwords stored in memory longer than necessary

### Privacy & Data Protection

- **Zero Knowledge**: PowerVault never transmits your data anywhere
- **Local Storage Only**: All data stays on your device
- **No Cloud Dependencies**: Fully offline operation
- **No Telemetry**: No usage tracking or analytics
- **Portable Structure**: Complete isolation from system registry

### Stealth Technology Details

**File Hiding Methods:**
1. **Windows Attributes**: Hidden + System flags
2. **Steganography**: Data embedded in carrier images
3. **Registry Tracking**: Encrypted stealth file registry
4. **Deep Hiding**: Multi-layer obfuscation

**Anti-Forensics:**
- Secure file deletion with multiple overwrites
- Metadata sanitization
- Timestamp normalization
- Slack space elimination

## üõ†Ô∏è Troubleshooting

### Common Issues

**Issue: Forgot Master Password**
- **Solution**: Master password cannot be recovered. If lost, data cannot be accessed. Use emergency recovery procedures if configured.

**Issue: Application Won't Start**
- Check .NET 8.0 Runtime is installed
- Run as Administrator
- Check antivirus isn't blocking PowerVault.exe
- Review crash logs in `CrashLogs` folder

**Issue: Backup Fails**
- Verify destination has sufficient space
- Check folder permissions
- Ensure no other process is accessing vault files
- Try different backup location

**Issue: Slow Performance**
- Reduce PBKDF2 iterations in security settings
- Disable GPU acceleration if causing issues
- Check available RAM and disk space
- Close unnecessary applications

**Issue: Stealth Files Not Hiding**
- Run PowerVault as Administrator
- Check Windows File Explorer settings (show hidden files)
- Verify NTFS file system (stealth requires NTFS)
- Review stealth registry for errors
